# Slack List Template — Project #1 Gate Tracker

Columns:
- Item ID
- Workstream (intake/arch/api/web/mobile/qa/release/security)
- Gate
- Status (Not Started / In Progress / GO / CONDITIONAL GO / NO-GO)
- Earliest Missing Gate? (Y/N)
- Owner
- Due Date
- Evidence Link
- Security Impact
- Decision Needed (Y/N)
- Notes

Default views:
1. Earliest Missing Gate = Y
2. Security Impact in (high, critical)
3. Decision Needed = Y
4. Due in next 7 days
