## Recommended v1 stack
- PostgreSQL
- Appsmith
- Metabase

## Primary user roles
- Owner
- Chief of Staff
- Main Orchestrator
- Security / Compliance Agent

## v1 goal
Provide a self-hosted visual interface for portfolio oversight across software and non-software work.

## Out of scope for v1
- public multi-tenant SaaS
- mobile-native app
- advanced resource forecasting
- billing
- external customer portal
