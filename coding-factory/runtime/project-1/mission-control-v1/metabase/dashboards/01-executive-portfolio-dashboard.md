# Dashboard Spec — Executive Portfolio Dashboard

## Purpose
Fast executive review of the entire portfolio.

## Cards
- active project count
- blocked project count
- due in 30 days
- owner decisions needed
- projects by category
- projects by health
- projects by priority
- recent status changes

## Data sources
- `vw_project_summary`
- `vw_blocked_projects`
- `vw_due_soon`
- `vw_owner_decisions`
- `project_status_history`
- `vw_portfolio_rollup`
