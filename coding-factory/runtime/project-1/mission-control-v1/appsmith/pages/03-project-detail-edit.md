# Appsmith Page Spec — Project Detail / Edit

## Goal
Show one project with related milestones, tasks, risks, decisions, and security status.

## Widgets
- project metadata card
- milestones table + add form
- tasks table + add form
- risks table + add form
- decisions table + add form
- security status panel

## Queries
- `qReadProjectBySlug`
- `qReadMilestonesByProject`
- `qReadTasksByProject`
- `qReadRisksByProject`
- `qReadDecisionsByProject`
- `qReadSecurityReviewByProject`
