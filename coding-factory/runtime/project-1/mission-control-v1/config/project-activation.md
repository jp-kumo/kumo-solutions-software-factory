# Project Activation — Mission Control v1

## Project name
Mission Control v1

## Goal
Create a self-hosted visual command center for all projects, priorities, milestones, decisions, and software security posture.

## In scope
- internal-only web experience
- PostgreSQL source of truth
- Appsmith operational pages
- Metabase dashboards
- weekly review support
- software project security tracking

## Out of scope for v1
- external users
- public deployment
- custom mobile app
- advanced RBAC
- custom React/Next.js front end
