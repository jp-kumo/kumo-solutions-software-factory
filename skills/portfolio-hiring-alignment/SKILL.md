# Portfolio Hiring Alignment

Use this skill to align output to hiring outcomes.

## Do this
- Map project features to business outcomes (cost, risk, speed, reliability).
- Convert implementation work into recruiter-facing bullets.
- Build STAR/CAR interview stories from shipped work.
- Match portfolio artifacts to target job descriptions.

## Output format
- For each project provide: employer problem, solution, measurable impact, stack, and role relevance.
