# Fiverr Offer Engineering

Use this skill to create high-conversion Fiverr service offerings.

## Do this
- Build offers around clear buyer outcomes, not tools.
- For each gig define: title, target buyer, pain point, deliverables, turnaround, revisions, exclusions.
- Create 3 tiers (Basic/Standard/Premium) with clear scope boundaries.
- Write concise descriptions with concrete benefits and trust signals.
- Add qualifying intake questions to reduce bad-fit buyers.

## Quality bar
- Each offer must be specific, measurable, and feasible in stated turnaround.
- Include one upsell/cross-sell path where relevant.
