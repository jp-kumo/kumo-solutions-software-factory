# Consulting Sales Pipeline

Use this skill to manage Jacques's freelance/consulting funnel.

## Pipeline stages
- Lead -> Qualified -> Discovery -> Proposal Sent -> Negotiation -> Won/Lost -> Delivery -> Testimonial.

## Do this
- Create qualification checklist (budget, timeline, problem urgency, decision maker).
- Use discovery questions that uncover business impact and success criteria.
- Draft proposal with scope, assumptions, timeline, deliverables, price, and next steps.
- Define follow-up sequence (48h, 4d, 7d) with concise value reminders.
- Capture lessons from won/lost deals.
