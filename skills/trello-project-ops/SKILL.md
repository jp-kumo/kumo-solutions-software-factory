# Trello Project Ops

Use this skill to run Jacques's project management system consistently.

## Do this
- Maintain lists: Inbox, Backlog, This Week, Today, In Progress, Blocked, Review/QA, Done, Wins Archive.
- Enforce WIP limits: Today max 3 cards, In Progress max 2 cards.
- Ensure each card has: why it matters, definition of done, deliverables, dependencies, estimate, next action.
- At day end: move cards, record blockers, create next actions.
- Weekly: produce wins summary and next-week priorities.

## Card title pattern
- `[Outcome] - [Concrete deliverable]`

## Output expectations
- Keep updates concise and execution-focused.
- Prioritize items that move hiring readiness and near-term income.
