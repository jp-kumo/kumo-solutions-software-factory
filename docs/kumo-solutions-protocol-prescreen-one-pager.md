# Kumo Solutions — AI Protocol Pre-Screen (One-Pager)

## What this is
A focused AI-assisted pre-screen service for clinical trial protocols that surfaces high-risk issues early, before full manual review cycles consume days.

## Problem we solve
Clinical teams lose time on first-pass protocol review because checks are scattered across:
- FDA / EMA / local guidance
- internal SOPs
- protocol design consistency

Result: delayed cycles, avoidable rework, and hidden risk.

## Our solution
**Kumo Solutions Protocol Pre-Screen** delivers a structured first-pass report in minutes, not days.

### Core outputs per protocol
1. **Compliance Gap Report** (FDA/EMA/local)
2. **SOP Conflict Report** (internal policy alignment)
3. **Clinical Consistency Flags** (endpoints, safety plan, population)
4. **Risk-ranked summary** (Critical / Major / Minor)
5. **Citation-backed findings** (source + section references)

## Delivery model
- Turnaround target: **15–20 minutes per protocol** after ingest
- Human-in-the-loop: all high-impact findings marked **"Needs reviewer decision"**
- Output format: executive summary + detailed appendix (Markdown/PDF)

## Design principles
- Single orchestrator for planning/control
- Specialist agents with single responsibility
- Explicit conflict rules (external regulations override internal SOP)
- Confidence scoring used as signal, not truth
