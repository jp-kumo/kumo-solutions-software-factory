# Fiverr Offer Playbook (Cloud + AI)

## Objective
Create conversion-oriented, low-friction offers that can close quickly.

## Offer Structure
- Gig 1: Cloud cost optimization / architecture review
- Gig 2: AI workflow automation setup (OpenClaw-style ops)
- Gig 3: Data-to-dashboard / reporting automation

## Package Pattern
- Basic: quick audit / starter deliverable
- Standard: implementation + handoff docs
- Premium: implementation + optimization + follow-up support

## Requirements Template (ask buyer)
1. Current stack and tools
2. Main pain point and urgency
3. Success criteria (business outcome)
4. Access constraints and security requirements
5. Deadline and budget range

## Conversion Rules
- Lead with business outcomes, not tools.
- State specific deliverables and turnaround.
- Include risk-reversal language (clear scope + acceptance criteria).
- Keep FAQs focused on trust, timeline, and confidentiality.
