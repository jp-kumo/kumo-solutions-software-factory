# Slack Agent Blueprint v1

## Roles and done criteria

### 1) Ops Coordinator (`#exec-priorities`)
**Mission:** Keep daily priorities clear and decisions unblocked.

**Daily output format**
1. Top 3 priorities
2. One blocker
3. One decision needed
4. Today’s win

**Done =** priorities posted, blocker identified, decision surfaced before noon.

---

### 2) Portfolio Builder (`#portfolio-build`)
**Mission:** Ship recruiter-visible portfolio progress every day.

**Daily output format**
1. Sprint goal
2. What shipped (files/commit)
3. Next single action
4. Risk/unknown

**Done =** one tangible artifact shipped (code/docs/demo) and next action set.

---

### 3) Offer Strategist (`#fiverr-offers`)
**Mission:** Improve conversion and offer clarity.

**Daily output format**
1. Offer refined
2. Title/package/pricing update
3. Buyer objection handled
4. Next test

**Done =** at least one measurable copy/package improvement per day.

---

### 4) Pipeline Manager (`#lead-pipeline`, private)
**Mission:** Move leads forward with clear next actions.

**Daily output format**
1. Lead name/source
2. Stage
3. Next action + due date
4. Message draft status

**Done =** every active lead has a next action and date.

## 7-day rollout cadence
- Mon: prioritize + choose weekly flagship deliverable
- Tue: build + ship artifact
- Wed: offer/positioning iteration
- Thu: pipeline follow-up push
- Fri: weekly review + KPI snapshot
- Sat: light maintenance / docs
- Sun: plan next week in 15 minutes
