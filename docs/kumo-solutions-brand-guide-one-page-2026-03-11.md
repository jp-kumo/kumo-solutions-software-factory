# Kumo Solutions - One-Page Brand Guide

## Brand type
Boutique consultancy

## Positioning
Kumo Solutions is a boutique consultancy helping regulated and high-trust organizations build secure cloud foundations and tailored AI governance solutions.

## Short brand line
**Secure Cloud & AI Governance**

## Brand pillars
- Clarity
- Trust
- Tailoring
- Practicality

## Brand personality
Calm, intelligent, precise, trustworthy, polished, thoughtful, modern, understated.

## Voice and tone
Use a voice that is:
- professional
- clear
- measured
- warm, but not casual
- confident, but not overblown

Prefer words like:
- secure
- governance
- tailored
- practical
- readiness
- high-trust
- regulated

Avoid words like:
- 10x
- growth hacks
- guru
- disruptive
- one-size-fits-all

## Color palette
- **Kumo Purple:** `#5224BA`
- **Kumo Orange:** `#FE8613`
- **Dark Gray:** `#4B525C`
- **Light Gray:** `#A1A5B0`
- **Soft Neutral Background:** `#F7F4F3`

Suggested usage ratio:
- 70% white / soft neutral
- 15% dark gray
- 10% purple
- 5% orange

## Typography
Recommended web font pairing:
- **Headings:** Manrope
- **Body:** Inter

Suggested web scale:
- H1: 44-56 px
- H2: 32-40 px
- H3: 24-28 px
- Body: 16-18 px
- Small label text: 12-14 px

## Logo system
- **Primary logo:** Current official Kumo Solutions logo
- **Approved tagline:** Secure Cloud & AI Governance
- **Secondary motif:** Kanji cloud concept for About sections, proposals, watermarks, and subtle branded accents

## UI style
The website should feel like a small premium consultancy:
- wide margins
- generous white space
- soft shadows
- rounded cards
- restrained motion
- clean section hierarchy
- a single prominent primary CTA

## Core CTA
**Book a Readiness Call**

## Brand test
Does this make Kumo Solutions feel like a small premium consultancy for serious buyers?

If yes, keep it. If it feels generic, flashy, or broad, cut it.
