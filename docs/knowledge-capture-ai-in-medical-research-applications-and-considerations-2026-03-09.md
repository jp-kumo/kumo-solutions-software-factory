## Practical implementation checklist (derived for applied work)
1. Define intended-use boundaries for each clinical/medical AI workflow.
2. Require model output traceability and evidence-grounded decision support.
3. Separate assistive recommendations from autonomous decision authority.
4. Implement bias/fairness checks across demographic and clinical subgroups.
5. Add privacy controls for sensitive health data (least privilege + encryption + retention policy).
6. Build legal/compliance review gates before deployment changes.
7. Add monitoring for drift, error rates, and adverse recommendation patterns.
8. Ensure clinician/human override in high-risk decision pathways.
9. Document cloud shared-responsibility controls and incident response paths.
10. Maintain versioned governance artifacts for audit readiness.
