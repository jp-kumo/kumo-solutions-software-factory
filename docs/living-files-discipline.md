# Living-Files Discipline

## Principle
If it matters later, it must exist in a file.

## Required Habits
1. Record decisions in markdown (not only in chat).
2. Keep long-term facts in `MEMORY.md`.
3. Keep daily progress in `memory/YYYY-MM-DD.md`.
4. Keep playbooks in `/docs` so work can be repeated.
5. Keep prompts short but persistent instructions in files.

## Decision Log Format
- Date
- Decision
- Why
- Impact
- Next action

## Trigger Rule
When we say “remember this,” update a file immediately.
