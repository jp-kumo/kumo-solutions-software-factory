# Focus Mode (ADHD-Friendly)

## Purpose
Reduce overwhelm and increase daily execution with simple, repeatable structure.

## Active Automations
1. **Daily Top 3** — 8:20 AM America/Chicago
2. **Midday Sprint Nudge** — 1:30 PM America/Chicago
3. **Evening Reset** — 9:40 PM America/Chicago

All run with low-cost settings:
- Model: `google/gemini-3-flash`
- Thinking: `minimal`

## Interaction Style
- One clear next action at a time
- 5-minute starter steps to lower activation friction
- Short prompts and low cognitive load
- Carry-over handling to avoid decision fatigue

## Manual Trigger Template
If Jacques asks for focus help on demand, respond with:
1. Top 1 priority right now
2. 5-minute starter action
3. 25-minute sprint objective
4. One anti-distraction rule
