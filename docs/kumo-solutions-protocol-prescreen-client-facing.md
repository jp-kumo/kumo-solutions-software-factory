# Kumo Solutions
## Protocol Pre-Screen AI (Client-Facing)

Speed up first-pass clinical protocol review with structured, citation-backed pre-screening.

---

### What you get
For each protocol, Kumo Solutions delivers:

1. **Compliance Gap Summary**
   - FDA / EMA / local framework checks
2. **Internal SOP Alignment Check**
   - policy conflicts clearly identified
3. **Clinical Design Consistency Flags**
   - endpoints, safety plan, population coherence
4. **Risk-ranked Findings**
   - Critical / Major / Minor
5. **Citation-backed Output**
   - source + section references for reviewer trust
